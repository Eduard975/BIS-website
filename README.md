# BIS 2026 - Official Website

This repository contains the source code for the **BEST Iasi Symposium 2026**.

### Core Frameworks

- **React:** `19.2.0` (Utilizing the React Compiler)
- **Vite:** `7.2.4` (Build Tool)
- **Tailwind CSS:** `4.1.18` (Styling)
- **TypeScript:** `5.9.3` (Type Safety)
- **React Router:** `7.12.0` (Navigation)

### Key Dependencies

- **Swiper** (`12.0.3`): For high-performance touch sliders and carousels.
- **React Icons** (`5.5.0`): Extensive UI icon library.
- **Flag Icons** (`7.5.0`): SVG collection of all world flags for international speakers.
- **React GA4** (`^2.1.0`): Google Analytics 4 integration for traffic tracking.

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed and **pnpm** as your package manager.

### Installation

Run to install required dependencies

```bash
pnpm install
```

### Available Scripts

`pnpm dev` -- Starts the Vite dev server.

`pnpm build` -- Compiles TypeScript and builds the production bundle.

`pnpm lint` -- Runs ESLint to find code quality issues.

`pnpm preview` -- Serves the generated production build for testing.
